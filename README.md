# Оценка функции конкурентоспособности для популяции с переменной численностью


### Этапы работы:
* Генерация выборки данных;
* Вывод функции конкурентоспособности с использованием методов машинного обучения;
* Проверка полученных результатов;
* Визуализация результатов.

### Используемая математическая модель
$$
z_1' = r z_1 + h_1 z_1^2 + s_1 z_2 z_1 (1 + b_1 z_1 + a_1 z_2) - s_2 z_2 z_1 (1 + b_2 z_2 + a_2 z_1) - z_1 (q z_1 + p z_2)
$$
$$
z_2' = r z_2 + h_2 z_2^2 + s_2 z_2 z_1 (1 + b_2 z_2 + a_2 z_1) - s_1 z_2 z_1 (1 + b_1 z_1 + a_1 z_2) - z_2 (q z_1 + p z_2)
$$
